# Job Search App

Одностраничное приложение для поиска работы.

Приложение разработано на Vue.js с использованием Vuex, Vue-router. Для сборки проекта использован Vue-cli.

- Данные для цветовой темы приложения сохраняются в localeStorage;
- Данные, используемые в компонентах хранятся в Store;

В проект добавлен следующий функционал:

1. Возможность смены цветового оформления;
1. Возможность фильтрации по названию вакансии, названию компании, расположению компании и варианту занятости (полная или частичная);
1. Динамическая подгрузка вакансий (при нажатии на кнопку "Load More");
1. Просмотр описания вакансии (при клике по названию вакансии);
1. Просмотр сайта работодателя (при клике по кнопке "Company Site" на странице описания вакансии);
1. Возврат на главную страницу (при клике по кнопке "Back" на странице описания вакансии).

Демо проекта можно посмотреть по этой [ссылке](https://dev-vacancies-app.netlify.app/)

##### Для установки зависимостей использовать:

```shell
npm install
```

##### Для запуска в режиме разработки использовать:

```shell
npm run serve
```

##### Для сборки в Production использовать:

```shell
npm run build
```
